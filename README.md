# CSE351 Network

Computer Network course repository for CSE351.

This repository contains programming assignments (`PA`) and homework (`hw`) completed during the course.

## Directory Overview

- `PA1` - Programming Assignment 1 files, source code, test samples, and report
- `PA2` - Programming Assignment 2 files, proxy implementation, tester, and report
- `PA3` - Programming Assignment 3 files, transport/network implementation, and report
- `PA4` - Programming Assignment 4 files and report
- `hw1` - Homework 1 materials and results
- `hw2` - Homework 2 materials, experiment data, figures, and report

## Notes

- `PA` stands for Programming Assignment
- `hw` stands for Homework
- Each folder includes the corresponding source files, experiment results, and submission documents
